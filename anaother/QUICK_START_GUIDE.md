# 🚀 দ্রুত শুরু গাইড (Quick Start Guide)

## ⚡ ৫ মিনিটে Pagination সেটআপ করুন

### ✅ ধাপ ১: WordPress থিম রিলোড করুন (যদি প্রথমবার হয়)

**WordPress Dashboard এ যান:**
```
1. Admin Panel খুলুন: yoursite.com/wp-admin/
2. "Appearance" মেনু খোলুন
3. "Themes" এ ক্লিক করুন
4. আপনার থিম খুঁজুন (আমরা "smart-e-hub" ব্যবহার করছি)
5. যদি অ্যাক্টিভ না থাকে তাহলে "Activate" বাটনে ক্লিক করুন
```

**কি ঘটবে:**
- থিমটি সক্রিয় হবে
- Pagination Rewrite rules স্বয়ংক্রিয়ভাবে সেট হবে

---

### ✅ ধাপ ২: Permalink সেটিংস যাচাই করুন

**WordPress Dashboard এ যান:**
```
1. "Settings" মেনু খোলুন
2. "Permalinks" এ ক্লিক করুন
3. যাচাই করুন যে:
   ☑️ "Post name" নির্বাচিত আছে
   বা
   ☑️ কোনো কাস্টম permalink ফরম্যাট নির্বাচিত আছে
   (নয়: "Plain" বা "Day and name")
4. নীচে "Save Changes" বাটনে ক্লিক করুন
```

**সাধারণ কনফিগারেশন:**
```
✅ OK: /%postname%/
✅ OK: /blog/%postname%/
✅ OK: /%year%/%postname%/
❌ NO: /?p=123 (Plain - এটি হবে না)
```

---

### ✅ ধাপ ৩: কমপক্ষে 13টি প্রোডাক্ট তৈরি করুন

**WordPress Dashboard এ:**
```
1. Left menu এ "Products" খোলুন
2. "Add New" বাটনে ক্লিক করুন
3. প্রোডাক্ট ডেটা পূরণ করুন:
   - শিরোনাম: আপনার প্রোডাক্ট নাম
   - ACF ফিল্ড: প্রোডাক্ট ইমেজ, প্রাইস ইত্যাদি
4. "Publish" বাটনে ক্লিক করুন
5. এই প্রক্রিয়া ন্যূনতম ১৩ বার পুনরাবৃত্তি করুন
```

**নোট:** 
- প্রথম 12টি প্রোডাক্ট পেজ 1-এ দেখা যাবে
- 13তম প্রোডাক্ট পেজ 2-এ দেখা যাবে
- এটি Pagination সক্রিয় করবে

---

### ✅ ধাপ ৪: পরীক্ষা করুন - সেটআপ সম্পূর্ণ! ✨

**ওয়েবসাইটে যান:**
```
1. আপনার সাইটের হোমপেজ খোলুন: yoursite.com/
2. প্রোডাক্ট সেকশন স্ক্রোল ডাউন করুন
3. পেজিনেশন দেখবেন (নিচে)
4. "Next" বাটনে ক্লিক করুন
5. পরবর্তী 12টি প্রোডাক্ট দেখা যাবে ✅
```

## 🎯 Pagination ব্যবহার করা

### ডেস্কটপে Pagination:
```
┌───────────────────────────────────────┐
│          প্রোডাক্ট সেকশন              │
│    (প্রথম 12টি প্রোডাক্ট দেখাবে)     │
└───────────────────────────────────────┘

          ← Previous  1  2  3  Next →

👉 কিভাবে ব্যবহার করবেন:
   • পরবর্তী প্রোডাক্ট দেখতে: "Next" বাটনে ক্লিক
   • পূর্ববর্তী প্রোডাক্ট দেখতে: "Previous" বাটনে ক্লিক
   • সরাসরি পেজ যেতে: পেজ নম্বর (2, 3, ইত্যাদি) ক্লিক করুন
```

### মোবাইলে Pagination:
```
┌──────────────────┐
│  প্রোডাক্ট সেকশন   │
│  (1 কলামে)       │
└──────────────────┘

   ← 1 2 3 →

👉 নোট: লেবেল ছোট জায়গার জন্য লুকানো
   • তীরগুলি ক্লিক করে পেজ নেভিগেট করুন
   • পেজ নম্বরে ক্লিক করে সরাসরি যান
```

## 🎨 কিভাবে দেখাবে - স্ক্রিনশট বর্ণনা

### Pagination বাটন ডিজাইন:

**Normal (অসক্রিয় পেজ):**
```
┌─────┐
│  2  │ ← হোয়াইট ব্যাকগ্রাউন্ড, নেভি টেক্সট
└─────┘    গ্রে বর্ডার
```

**Hover (মাউস ওভার):**
```
╔═════╗
║  2  ║ ← নেভি ব্লু ব্যাকগ্রাউন্ড
╚═════╝    সাদা টেক্সট, সাডো যুক্ত
↑ একটু উপরে উঠে যাবে
```

**Current (বর্তমান পেজ):**
```
╔═════╗
║  2  ║ ← টার্কুইজ গ্রেডিয়েন্ট ব্যাকগ্রাউন্ড
╚═════╝    সাদা টেক্সট, শক্তিশালী সাডো
```

## 🛠️ সাধারণ সেটআপ সমস্যা এবং সমাধান

### ❓ সমস্যা: Pagination দেখা যাচ্ছে না

**কারণ ১: পর্যাপ্ত প্রোডাক্ট নেই**
```
✅ সমাধান: কমপক্ষে 13টি প্রোডাক্ট তৈরি করুন
```

**কারণ ২: Permalink সেটিংস ভুল**
```
✅ সমাধান: 
   1. Settings → Permalinks খুলুন
   2. "Post name" বা কাস্টম permalink নির্বাচন করুন
   3. "Save Changes" ক্লিক করুন
```

**কারণ ৩: Theme সঠিকভাবে সক্রিয় নয়**
```
✅ সমাধান:
   1. Appearance → Themes
   2. আপনার থিম খুঁজুন
   3. "Activate" বাটনে ক্লিক করুন
```

---

### ❓ সমস্যা: "Next" বাটনে ক্লিক করলে 404 ত্রুটি

**কারণ: Rewrite rules সেট হয়নি**
```
✅ সমাধান:
   1. Dashboard থেকে লগ আউট করুন
   2. আবার লগ ইন করুন
   3. এটি Rewrite rules রিসেট করবে
```

**বিকল্প সমাধান:**
```
1. Settings → Permalinks
2. বর্তমান ফরম্যাট নিচে স্ক্রোল করুন
3. "Save Changes" ক্লিক করুন
4. হোমপেজ রিফ্রেশ করুন
```

---

### ❓ সমস্যা: Pagination স্টাইলিং কাজ করে না (রঙ দেখা যায় না)

**কারণ: ব্রাউজার ক্যাশ**
```
✅ সমাধান:
   • Windows/Linux: Ctrl + Shift + Delete
   • Mac: Cmd + Shift + Delete
   → All time / All period নির্বাচন করুন
   → "Clear browsing data" ক্লিক করুন
```

**অন্য সমাধান:**
```
1. ব্রাউজার বন্ধ করুন
2. আবার খুলুন এবং সাইট রিফ্রেশ করুন (F5)
3. CSS এখন সঠিক হওয়া উচিত
```

'posts_per_page' => 10,  ← প্রতি পেজে 10টি প্রোডাক্ট
