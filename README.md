# Digital E-Product BD Marketplace

A clean, premium static marketplace page inspired by the provided reference screenshot. The project uses semantic HTML, modular CSS, and vanilla JavaScript only.

## Structure

```text
/
├── index.html
├── css/
├── js/
│   └── app.js
├── assets/
│   ├── bana
│   ├── images/
│   └── icons/
└── README.md
├── style.css
└── responsive.css
```

## Features

- Sticky responsive header with search, cart badge, menu toggle, and primary CTA.
- Dark hero title section matching the original page hierarchy.
- Responsive product grid: 3 columns desktop, 2 columns tablet, 1 column mobile.
- Modern product cards with hover lift, image zoom, pricing hierarchy, and accessible buttons.
- Four-column footer with contact details, links, social icons, and newsletter form.
- Vanilla JavaScript for sticky header state, mobile actions, search filtering, and button feedback.

## Run

Open `index.html` in a browser. No build step is required.
